Privacy Policy

Last updated: January 3, 2026

Overview

Toonify MCP is a completely local, open-source MCP (Model Context Protocol) plugin. Your privacy is paramount - we designed Toonify to keep all data processing under your control.

Data Processing

100% Local Processing: Toonify operates entirely on your machine. All token optimization happens locally:

  • TOON format conversion (local)
  • Caching (local LRU + Claude prompt cache)
  • Statistics tracking (local)

You own your data: Toonify never sends your data, prompts, or statistics to external servers. Everything stays on your computer.

Claude API: When Toonify optimizes tool responses, the optimized output is passed to Claude through your existing Claude Code subscription. This is governed by Anthropic's Privacy Policy.

Website (toonify.pcircle.ai)

No Analytics: Our documentation website does not use tracking, analytics, or cookies.

Google Fonts: The website uses Google Fonts for typography. Google may collect your IP address as part of font delivery. See Google Privacy Policy.

Open Source Transparency

Toonify is MIT-licensed open source. You can:

  • Inspect all source code at GitHub
  • Verify that no data is sent externally
  • Modify the code to suit your needs
  • Use it freely in commercial or non-commercial projects

Your Control

  • All processing happens on your machine
  • No telemetry or usage tracking
  • Complete source code transparency - view on GitHub
