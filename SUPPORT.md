# Support

## What this repo is for

`toonify-mcp` is a public repository for the MCP server, Claude Code plugin, and supporting docs for automatic token optimization.

## Get help

### Usage and setup questions

Open a GitHub issue if you need help with:

- installation
- plugin or MCP setup
- configuration
- expected optimization behavior
- cache behavior

### Bug reports

When reporting a bug, include:

- your OS
- Node.js version
- Claude Code or MCP client context
- what you expected
- what happened instead
- a minimal reproducible example when possible

### Security issues

Do not file public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).
