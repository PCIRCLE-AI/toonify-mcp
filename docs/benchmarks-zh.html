Toonify MCP 結構化資料 Benchmark 摘要

這一頁整理的是 Toonify 目前的結構化資料 benchmark 摘要，方便你快速看目前量測結果、如何重跑，以及它在整體產品中的位置。

STRUCTURED DATA SUITE
12 個案例

包含 small、medium、large、nested、多語言、CSV-like 與 API-shaped payload。

LOCAL VERIFICATION RUN
48.1%

來自 2026 年 4 月 30 日本機驗證執行的平均節省值。

OBSERVED RANGE
24.5-66.3%

同一次執行中觀察到的最小到最大節省範圍。

如何重現

上述數字來自以下這個已提交在 repo 中的 Jest benchmark 測試套件。

NODE_OPTIONS=--experimental-vm-modules npx jest tests/benchmarks/quick-stats.test.ts --runInBand
打開 benchmark 測試套件 打開延伸 token benchmark 打開 code compression benchmark 原始碼
