Key Features

Built for developers who care about efficiency

🎯

Automatic Optimization

PostToolUse hook intercepts supported tool results and applies optimization automatically in plugin mode.

claude plugin add toonify-mcp
⚡

Structured Data Benchmarks

The current benchmark suite in `tests/benchmarks/quick-stats.test.ts` produced 48.1% average savings across 12 structured-data fixtures.

🌍

Code Compression Support

Supports TypeScript, Python, and Go source-code compression through comment and whitespace reduction in the pipeline.

🔧

Caching

Includes LRU result caching with TTL expiration and optional disk persistence.

🔄

Dual Mode

Works as Plugin (automatic) or MCP Server (manual). Choose the mode that fits your workflow.

📊

Built-in Metrics

Track token savings locally and inspect optimization statistics through MCP tools.

How It Works

Transparent optimization in Plugin mode

1

User reads large JSON file

Claude Code calls Read tool to fetch file contents

2

PostToolUse hook intercepts result

Hook automatically checks if content should be optimized

3

TOON format conversion

JSON/CSV/YAML converted to compact TOON representation

4

Optimized content sent to Claude API

Structured data is compressed when estimated savings clear the configured threshold.

Installation

Start with the default plugin path. Use MCP server mode only if you need manual control.

Recommended: Plugin Mode

Best for Claude Code users who want automatic optimization with no manual tool calls.

# Download the repository git clone https://github.com/PCIRCLE-AI/toonify-mcp.git cd toonify-mcp # Install deps and build npm install npm run build # Install globally from local source npm install -g . # Add as plugin (automatic mode) claude plugin add toonify-mcp # Verify installation claude plugin list # Should show: toonify-mcp ✓

Result: PostToolUse hook optimizes supported structured data and source-code results automatically.

Advanced: MCP Server Mode

Use this when you want explicit control or need to connect Toonify to another MCP client.

# Prerequisite: complete Option A to install toonify-mcp # Register as MCP server claude mcp add toonify -- toonify-mcp # Verify claude mcp list # Should show: toonify: toonify-mcp - ✓ Connected

For explicit control or non-Claude Code MCP clients. Call tools manually when needed.
