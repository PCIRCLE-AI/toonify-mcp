最後更新：2026 年 1 月 3 日

1. MIT 授權

Toonify MCP 是根據 MIT 授權 的免費開源軟體。

您可以自由使用、修改和分發 Toonify，包括用於商業目的，但需遵守 MIT 授權條款。

2. 無保證聲明

Toonify 以「現狀」提供，不提供任何明示或默示的保證。我們不保證：

  • Toonify 將無錯誤或不中斷運行
  • Token 優化將達到特定的節省百分比
  • Toonify 將適用於所有 MCP 工具或配置
  • 與未來 Claude Code 或 MCP 協議版本的相容性

3. 責任限制

Toonify 開發者和貢獻者不對您使用 Toonify 造成的任何損害負責，包括但不限於：

  • Claude API 產生的意外 token 成本
  • 資料遺失或損壞
  • 效能問題或優化失敗
  • 與其他工具的相容性問題
  • 快取相關問題

4. Token 優化

Toonify 會自動將結構化資料（JSON、CSV、YAML）轉換為 TOON 格式以優化 token 使用。您確認：

  • Token 節省量會根據資料結構和內容而異
  • TOON 格式轉換是透明且可逆的
  • 啟用時會自動進行優化
  • 您可以隨時停用優化

5. 您的責任

使用 Toonify 時，您同意：

  • 驗證 TOON 優化後的輸出符合您的需求
  • 使用 Claude API 時遵守 Anthropic 的可接受使用政策
  • 修改或分發 Toonify 時尊重 MIT 授權條款
  • 不將 Toonify 用於非法或有害目的
