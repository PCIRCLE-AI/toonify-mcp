安裝方式

預設先走外掛模式。只有在需要手動控制時再使用 MCP 伺服器模式。

⭐

推薦：外掛模式

適合 Claude Code 用戶。安裝後自動優化，不需要手動呼叫工具。

# 1. 下載專案
git clone https://github.com/PCIRCLE-AI/toonify-mcp.git
cd toonify-mcp

# 2. 安裝依賴並建置
npm install
npm run build

# 3. 從本機安裝
npm install -g .

# 4. 新增為外掛（自動模式）
claude plugin add toonify-mcp

# 5. 驗證安裝
claude plugin list
# 應顯示：toonify-mcp ✓
🔧

進階：MCP 伺服器模式

適合需要明確控制或要接到其他 MCP 客戶端的情境。

# 1. 先完成外掛模式安裝，確保 toonify-mcp 可用

# 2. 註冊為 MCP 伺服器
claude mcp add toonify -- toonify-mcp

# 3. 驗證
claude mcp list
# 應顯示：toonify: toonify-mcp - ✓ Connected
