最後更新：2026 年 5 月 1 日

概述

這一頁說明的是公開網站與開源 Toonify MCP 專案的使用邊界。它的目的是補充說明，不會取代實際規範軟體授權的 MIT 授權。

1. MIT 授權

Toonify MCP 是根據 MIT 授權 的免費開源軟體。

您可以自由使用、修改和分發 Toonify，包括用於商業目的，但需遵守 MIT 授權條款。

2. 無保證聲明

Toonify 以「現狀」提供，不提供任何明示或默示的保證。我們不保證：

  • Toonify 將無錯誤或不中斷運行
  • 上下文壓縮將達到特定的節省百分比
  • Toonify 將適用於所有 MCP 工具或配置
  • 與未來 Claude Code 或 MCP 協議版本的相容性

3. 責任限制

Toonify 開發者和貢獻者不對您使用 Toonify 造成的任何損害負責，包括但不限於：

  • Claude API 產生的意外 token 成本
  • 資料遺失或損壞
  • 效能問題或優化失敗
  • 與其他工具的相容性問題
  • 快取相關問題

4. 上下文壓縮

啟用時，Toonify 會自動壓縮支援的結構化資料、debug 輸出與支援的原始碼。您確認：

  • 節省量會根據資料結構、debug 輸出、原始碼與內容型態而異
  • 結構化資料可能會轉換為 TOON 格式
  • 支援的輸出可能會在工作流程往後傳遞前先被轉換
  • 只有在對應模式啟用時，才會自動進行支援的壓縮
  • 您可以隨時停用優化

5. 您的責任

使用 Toonify 時，您同意：

  • 自行判斷優化後的輸出是否適合您的使用情境
  • 在 production、合規或其他敏感流程中使用前，先驗證重要結果
  • 遵守您搭配 Toonify 使用的 AI 供應商或 client 的條款、隱私政策與使用規範
  • 修改或分發 Toonify 時尊重 MIT 授權條款
  • 不將 Toonify 用於非法或有害目的

6. 第三方服務

  • Claude Code： 受 Anthropic 適用的 Claude Code 法務與合規條款 約束

  • 其他 MCP client 或模型供應商： 受各自的條款、隱私政策與服務行為約束

7. 貢獻

一般情況下，對 repository 的程式碼與文件貢獻，會依照專案現有授權處理，除非 repository 另有說明。請參閱我們的 貢獻指南。
