運作原理

1

讀取檔案

Claude Code 呼叫 Read 工具讀取 JSON、CSV 或 YAML 檔案。

2

自動攔截

PostToolUse hook 自動攔截工具結果。

3

格式轉換

檢測資料類型並轉換為 TOON 格式。

✓

節省 Token

優化後的內容傳送到 Claude API，實現 ~48% 的 token 減少。

安裝方式

⭐

外掛模式（推薦）

自動 token 優化，無需手動呼叫

# 1. 下載專案
git clone https://github.com/PCIRCLE-AI/toonify-mcp.git
cd toonify-mcp

# 2. 安裝依賴並建置
npm install
npm run build

# 3. 從本機安裝
npm install -g .

# 4. 新增為外掛（自動模式）
claude plugin add toonify-mcp

# 5. 驗證安裝
claude plugin list
# 應顯示：toonify-mcp ✓
🔧

MCP 伺服器模式

需要手動控制或使用其他 MCP 客戶端

# 1. 先完成外掛模式安裝，確保 toonify-mcp 可用

# 2. 註冊為 MCP 伺服器
claude mcp add toonify -- toonify-mcp

# 3. 驗證
claude mcp list
# 應顯示：toonify: toonify-mcp - ✓ Connected
