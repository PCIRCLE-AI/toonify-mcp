隱私權政策

最後更新：2026 年 1 月 3 日

概述

Toonify MCP 是一個在本機運行的開源 MCP 伺服器與 Claude Code 外掛。您的隱私至關重要，因此產品設計的原則是讓支援的處理流程都留在您的掌控之下。

資料處理

100% 本機處理： Toonify 完全在您的電腦上運行。支援的壓縮流程與相關本機功能都在本機進行：

  • 支援內容的 TOON 格式轉換
  • 本機快取與 prompt cache 協作
  • 本機統計追蹤

您擁有您的資料： Toonify 從不將您的資料、提示或統計資訊傳送到外部伺服器。一切都保留在您的電腦上。

Claude API： 當 Toonify 優化工具回應時，優化後的輸出會透過您現有的 Claude Code 訂閱傳遞給 Claude。這受 Anthropic 隱私權政策 規範。

網站 (toonify.pcircle.ai)

無分析工具： 我們的文件網站不使用追蹤、分析或 Cookie。

Google Fonts： 網站使用 Google Fonts 顯示文字。Google 可能會收集您的 IP 位址作為字型傳送的一部分。請參閱 Google 隱私權政策。

開源透明

Toonify 使用 MIT 授權開源。您可以：

  • 在 GitHub 檢視所有原始碼
  • 驗證沒有資料被傳送到外部
  • 修改程式碼以符合您的需求
  • 在商業或非商業專案中自由使用

您的控制權

  • 所有處理都在您的電腦上進行
  • 無遙測或使用追蹤
  • 完整的原始碼透明度 - 在 GitHub 查看
